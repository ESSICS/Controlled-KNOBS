# Controlled-KNOBS
A MIDI controlled version of [KNOBS](https://github.com/ESSICS/KNOBS).

Controlled-KNOBS can be controlled by a [Midi Fighter Twister](https://store.djtechtools.com/products/midi-fighter-twister).
